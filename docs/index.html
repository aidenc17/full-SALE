üìã Project Overview

DegreeAdmin

A comprehensive degree planning system that enables students to select multiple majors and minors, visualize their academic pathway, and determine if they can complete their degree requirements within their desired timeframe. The system facilitates collaboration between students, faculty advisors, and administrators to create optimized academic schedules.

üéØ Key Features

Academic Planning

Students can select from a minimum of 2 majors (Economics, Business, Entrepreneurship, Marketing) and 2 minors, with the system automatically calculating prerequisite courses, identifying which faculty teach each course, and determining course offering schedules. The intelligent planning engine helps students visualize if they can complete all requirements within their target graduation date.

Secure Authentication

All users authenticate with secure credentials:

  • Usernames: 8 characters, lowercase only
  • Passwords: 12 characters with at least 1 uppercase, 1 lowercase, 1 numeric, and 1 symbol

Role-Based Access

The system provides tailored interfaces for each user type after authentication, ensuring students, faculty, and administrators see only the tools and information relevant to their role.
